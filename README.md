# origami-tests

Testy do zadania origami.
W rozwiązaniu niedokładności powinny być rozstrzygane na korzyść punktu **na linii**
i na korzyść punkty **w figurze**.

W random część losowych całkowicie (1-5000), a część losowych
ale papiery to kartka nr 1 zagięta $n-1$ razy, i zapytania tylko do ostatniej kartki (5001-10000).
